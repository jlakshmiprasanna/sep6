5.Difference between html and html5

-->New Semantic Elements

HTML4:Lacked many semantic elements.Often used div and span for everything

HTML5:Introduced elements like header,footer,section,article,aside.

-->Multimedia Support

HTML4: Relied on external plugins (like Flash) for audio and video content.

HTML5:Introduced native audio and video elements, allowing easy embedding of multimedia without plugins.

-->New Form Elements and Input Types

HTML4: Had basic form elements like input type, textarea, select

HTML5: Added new input types like input type="email", input type="date",input type="range" and attributes like required,autofocus and placeholder.

-->Depcated Elements and Attributes

HTML4:Included many outdated or redundant elements like font, center, and attributes like align for styling.

HTML5:Depcated many of these elements,pushing for the use of CSS for styling and layout.

-->Canvas and SVG Support

HTML4: Had limited support for drawing graphics; typically relied on images or external tools like Flash.

HTML5: Introduced the canvas element for drawing graphics and animations directly in the browser, and also improved support for SVG (Scalable Vector Graphics).

13.What is difference between Absolute and relative path?

Absolute path Specifies the full location of a file or directory from the root directory. In Windows, it starts with the drive letter, and in Unix-based systems, it starts with a forward slash. Absolute paths always point to the same location, regardless of the current directory.

Relative path Specifies the location of a file or directory in relation to the current working directory. Relative paths may point to different locations depending on the current directory.

4.How many types of selectors in css we have?

-->Simple Selector :- A simple selector is either a type selector or universal selector followed immediately by zero or more attribute selectors, ID selectors

-->Combinator Selector :- A combinator is something that explains the relationship between the selectors. A CSS selector can contain more than one simple selector.

-->Psuedo Classes :- A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected elements. Ex:- hover,active,enable,disabled,focus.

-->Psuedo Elements :- A CSS pseudo-element is used to style specified parts of an element. For example, it can be used to: Style the first letter, first line, After, Before.

-->Attribute Selector :- A CSS attribute selector matches elements based on the element having a given attribute explicitly set, with options for defining an attribute value or substring value match.

6.What is the difference between display:block and display:inline?

display:block An element that has the display property set to block starts on a new line and takes up the available screen width.You can specify the width and height properties for such elements.

display:inline An element with a display property set to inline will not start on a new line and it will take up the remaining/available screen width.Because of this, you can't set the width and height of an element that has a display of inline.

7.What is difference between float and text-align?

Text-align: An use align to align text and other items rather it be left, right, centered, or justified. Align DOES NOT remove the item from the document flow.

8.What is difference between Psuedo-classes and pseudo-element?

pseudo-class: A pseudo-class is a selector that selects elements that are in a specific state, e.g. they are the first element of their type, or they are being hovered over by the mouse pointer.

pseudo-element: Pseudo-elements behave in a similar way. However, they act as if you had added a whole new HTML element into the markup, rather than applying a class to existing elements.

11.What is difference between position:relative and position:absolute?

position:absolute: Positions an element based on the top and left margins of the page, relative to its closest positioned ancestor. This allows the element to be placed exactly where you want it, even on top of other elements.

position:absolute: Positions an element relative to its original position within the parent element. This means that setting the top, right, bottom, and left properties will move the element away from its normal position.

13.What is difference between transitions and transforms?

Transition : A process of changing from one state to another, which can be large or small, simple or complex. For example, a minor change in working practice is a transition.

Transform : Transform property in CSS moves or modifies the appearance of an element, whereas the Transition property seamlessly and gently transitions the element from one state to another.

14.What is difference between duration and timing function?

The animation-duration property specifies how long an animation takes to complete, in seconds or milliseconds. A value of 0s means no transition will occur, and the animation will switch between states instantly.

The animation-timing-function property controls how an animation accelerates and decelerates during its cycle. This property affects how the user perceives the animation's speed, and can make an animation start slowly, accelerate, maintain a constant speed, or end instantly.
